# Kısayollar

Quick reference for Minecraft Java Edition on keyboard/mouse:

Movement:

- W/A/S/D — move forward/left/back/right
- Space — jump (double-tap to fly in creative, hold/release to go up/down while flying)
- Shift — sneak / fly down in creative

Interaction:

- Right click — place block / use item
- Left click — break block / attack
- Q — drop held item

Inventory & Items:

- E — open inventory
- 1-9 — select hotbar slot
- Scroll wheel — cycle hotbar slots

TNT specific:

- Place TNT with right click
- Light it with flint and steel (right click on the TNT while holding it)
- Or use a redstone signal (lever, button, etc.)

Other useful:

- T or / — open chat / commands
- F3 — debug screen
- Esc — pause menu
- F5 — toggle camera perspective

To test your TNT: /give @s flint_and_steel, place a TNT block, then right-click it with the flint
and steel and run!
